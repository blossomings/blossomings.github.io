Simple Spanish Syllables | M, P, L, S

Learning Objectives

By the end of this lesson, you will be able to
➧ Recognize the letters M, P, L, and S.
➧ Pronounce each letter correctly.
➧ Blend these consonants with the five Spanish vowels.
➧ Read simple Spanish syllables.
➧ Read your first Spanish words.
➧ Read short, easy sentences.


Review

Before starting, let's review the five Spanish vowels.








Introduction

Great job learning the Spanish vowels!
Now it's time to add some consonants. By combining consonants with vowels, you'll be able to read many simple Spanish words.
Today you'll learn four important letters
